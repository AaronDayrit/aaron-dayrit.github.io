WEB DEVELOPMENT:

ALL ABOUT THE BASICS

DECEMBER 8, 2021 - AARON DAYRIT

Web Development is the process of creating a website on the internet for people to access. Websites you often visit go through the cycle of Web Development. They don't just suddenly exist from thin air. Web development can range from simple static text pages to complex applications for electronic businesses.

HTML

HTML (Hypertext markup language) is a programming language. HTML places text, photos, lists on a website. Styling a website is not the goal of HTML. You can think of it as the skeleton of a website, only placing the pieces or bones of what you want on a website.

HTML Elements

HTML elements are those pieces you want to place on a website. They are the text, photos, lists on a website you see. Two tags open and close a piece of HTML content to house a piece of webpage content.

-HTML element

HTML Attributes

HTML attributes modify aspects of HTML elements. Attributes can commonly be the URLs of links or buttons or the source of an image you see in a gallery.

-Image element attribute

BLOCK Elements

Block Elements are elements that, when created, are placed on a new line from a previous element.

-How a block element reacts

INLINE Elements

Inline Elements are elements that, when created, are placed on the same line as the element before the one you just created.

-How a inline element reacts

CSS

CSS (cascading style sheets) is another programming language for web development. Its primary purpose is to give styling to the elements that are on your website. It controls colors, font-sizing, how a website is structured, etc.

CSS Rule

A CSS rule groups one or more CSS styling properties to an HTML element. The rule specifies the target of what you want to style (EX. A paragraph), and how you want to style the element (EX. Font-size).

CSS Selector

Your CSS selector is the target of what you are trying to style. A selector could be a paragraph, or an image that you are trying to style.

-CSS selector format

CSS Property

A CSS property is the styling you want to give an HTML element. A property can be your font size for a paragraph or a width of an image.

-Property of H1 element

CSS Value

CSS properties have CSS values. It can change the property's amount or specific aspect that is unique to that property. An example can be the width-value of an image. A unique value for an image can be covering a section with your image element (object-fit: cover).

-Value of color attribute

Web Development is not an easy task. Mastering and remembering these concepts are the big stepping stones to mastering Web Development.
